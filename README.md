# LCEFloatingIslands
A mod for the Minecraft LCE leak that causes floating islands with bonus loot chests to spawn
![screenshot](https://github.com/swagpuppy420/LCEFloatingIslands/blob/d0d77bb63fd43d3210dd61661562ca909732aa95/Screenshot_20260301_233453.png)

# How to use 
Overwrite these files in the Minecraft.World directory of your leaked source code (not included here). After compilation, floating islands should spawn. You also have to edit one of the header files to include FloatingIslandFeature.h
